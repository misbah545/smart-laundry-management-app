# Smart Laundry Management App
MSCIT SEM 2 PROJECT

## About
This is a comprehensive Smart Laundry Management application built with **Flutter** (Frontend) and **Django** (Backend).

### Getting Started

#### Flutter Frontend
1. Navigate to the root directory.
2. Run `flutter pub get`
3. Run `flutter run`

#### Django Backend (in `django_backend/`)
1. Navigate to `django_backend/`.
2. Activate the virtual environment: `source venv/bin/activate`.
3. Run `python manage.py runserver`.
