Terminal 1 - Start Redis:
  $ redis-server

Terminal 2 - Start Django Backend:
  $ cd /home/root123/SmartLaundry
  $ /home/root123/SmartLaundry/venv/bin/daphne -b 0.0.0.0 -p 8000 backend.asgi:application

Terminal 3 - Start Frontend:
  $ cd /home/root123/SmartLaundry/frontend/smart_laundry_mobile_react
  $ npm start

Then open Expo Go app and scan the QR code.

ARCHITECTURE OVERVIEW

└── Enhanced Models (ImageFields, ETA fields)

Real-Time Layer (Redis)
├── chat.room_* (chat messages)
├── order.updates.* (order status)
└── location.* (driver position)

Data Layer (PostgreSQL)
├── messaging_message (chat messages)
├── messaging_chatroom (conversations)
├── tracking_trackinganalytics (order metrics)
├── tracking_geofenceevent (zone crossings)
├── tracking_analyticssnapshot (daily stats)
└── orders_order (enhanced with proofs, ETA)
